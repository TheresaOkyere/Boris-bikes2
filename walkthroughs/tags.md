* adjust model to ensure tags are uniq


```ruby
def tag_names=(tag_names)
  return nil if tag_names == ''
  tag_names.split(', ').uniq.each do |tag_name|
    formatted_name = '#'+tag_name.delete('#')

    tag = Tag.find_or_create_by(name: formatted_name)
    tags << tag
  end
end

```
